Advance Web Development: Day 3

SVG

SVG stands for Scalable Vector Graphics and is not pixel-based, so it can be magnified infinitely with no loss of quality. There are two ways to use SVG in webpages.
  • Directly include the image in svg file format: <img src="sample.svg" alt="Sample SVG File">
  • Create a SVG element tag with two attributes - width and height: <svg width="500" height="500"></svg>
To create various shapes in SVG we use different tags.

Animations

SVG animations can be created using the <animate /> tag. Note: It does not have a closing tag.

Path

SVG elements and shapes can also be created using the <path /> tag. Note: It does not have a closing tag.

Canvas

Canvas is used to draw graphics that include everything from simple lines to complex graphic objects. Syntax: <canvas id="canvas_area" width="500" height="500"></canvas>

Canvas element is only a container for graphics. Hence we must use a script to actually draw the graphics (usually JavaScript).
