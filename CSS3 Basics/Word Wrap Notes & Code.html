Word Wrap

The word-wrap property allows long words to be broken and wrapped into the next line. It takes two values:
  • word-wrap: break-word; the browser breaks a word when it is too long to fit within its container.
  • word-wrap: normal; No changes are made

Example

Example for word-wrap: normal; wsedfgbhnjmk,mjhgtfresdfrghnjmkjnhbgfrdewsedrftghyjkmjhgtfrderfghjkmjhygtfrerftgyhjjhygtfreddrfgh
Example for word-wrap: break-word; wsedfgbhnjmk,mjhgtfresdfrghnjmkjnhbgfrdewsedrftghyjkmjhgtfrderfghjkmjhygtfrerftgyhjjhygtfreddrfgh
