Vendor Prefixes

CSS vendor prefixes or CSS browser prefixes are a way for browser makers to add support for new CSS features during periods of testing and experimentation. These are different for different browsers.
  • Firefox: -moz-
  • Safari: -webkit-
  • Chrome: -webkit-
  • Opera: -o-
  • Internet Explorer/ Edge: -ms-

Example

-webkit-border-radius: 24px;
