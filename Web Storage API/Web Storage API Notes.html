Advance Web Development: Day 1

Web Storage API

Two types of web storages:
  • sessionStorage(): destroyed once the browser is closed
  • localStorage(): has no expiration date

Local Storage

The data is stored as key/value pairs.
  • Storing a Value: localStorage.setItem("key", "value");
  • Fetching a Value: localStorage.getItem("key");
  • Removing a Value: localStorage.removeItem("key");
  • Removing All Values: localStorage.clear();

Session Storage

The syntax for session storage is similar to that of Session Storage. The data is stored as key/value pairs.
  • Storing a Value: sessionStorage.setItem("key", "value");
  • Fetching a Value: sessionStorage.getItem("key");
  • Removing a Value: sessionStorage.removeItem("key");
  • Removing All Values: sessionStorage.clear();
