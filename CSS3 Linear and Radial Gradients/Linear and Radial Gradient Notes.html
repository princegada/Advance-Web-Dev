Advance Web Development: Day 5

Linear Gradient

To create a linear gradient, you must define at least two color stops. Color stops are the colors among which you want to render smooth transitions. You can also set a starting point and a direction - or an angle - along with the gradient effect.

Syntax: background: linear-gradient(colour_Stop_1, colour_Stop_2);

Colour Stops

Colors can be added one after the other, separated with a comma. Syntax: background:linear-gradient(colour 1, colour 2, colour 3...);

Color stop positions can be specified for each color. Syntax: background:linear-gradient(colour_1 20%, colour_2 30%, colour_3 85%);
Colour stops can be also be written in px, em, etc.

Direction of the Gradient

Syntax: background:linear-gradient(direction, colour 1, colour 2, colour 3...);
Directions supported are: left, right, top, bottom. We can also add combinations like bottom right, etc.

Angle of the Gradient

Syntax: background:linear-gradient(100deg, colour 1, colour 2, colour 3...);

Repeating a Linear-Gradient

The repeating-linear-gradient() function is used to repeat a linear gradient: Syntax: background:repeating-linear-gradient(blue, green 20px);

Radial Gradient

To create a radial gradient, you must define at least two color stop and are defined by its center.

Syntax: background: radial-gradient(position, shape or size, color-stops);

The first value defines the gradient position. We can use a descriptive keyword, such as top, bottom, center, or left; or we can specify, for example, 50% 50% to set the gradient at the center or 0% 0% to set the gradient to start at top left.

Values supported for shape or size are: ellipse(default) and circle
